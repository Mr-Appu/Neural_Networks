# Artificial Neural Network [ANN]
An attempt to build a whitebox and blackbox implementation of Atrificial Neural Networks to undersatnd how ANN learns.

# Data

  | $x_{1}$ | $x_{2}$ | y |
  | -- | -- | - |
  | 1  | 0  | 1 |
  | 1  | 1  | 1 |
  | 0  | 1  | 0 |
  | 0  | 0  | 0 |

  2 Input variables $x_{1}$ , $x_{2}$<br>
  1 Output variable y
